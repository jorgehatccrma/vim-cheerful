Section 1: Usage                                     *CheerfulUsage*

Currently this plugin provides 2 themes off-the-shelf: `bolero_light` and
`bolero_dark`. They can be applied as follows:

  `:colorscheme bolero_light`

or

  `:colorscheme bolero_light`

The function `BoleroToggle` is provided as a shortcut to alternate between
themes. Use it as follows:

  `:call BoleroToggle()`
